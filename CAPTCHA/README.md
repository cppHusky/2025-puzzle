# CAPTCHA

本关将展示一个验证码，玩家需要在16张图片中选出符合要求的所有图片（保证至少有一张图片符合要求）。

本关设计了若干个可选题目，这些题目随机出现。玩家可以换题（也是随机的）。

玩家只需通过任意一道可选题目，即可通过本关。

## 设计指引

每个题目所需要的 16 张图片均位于对应的文件夹下，以 `part_${r}_${c}.png` 命名，代表这个部分应位于第 r 行，第 c 列。（记左上角为第 1 行，第 1 列）
