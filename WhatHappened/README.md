## 交互方式

提供一个输入框，玩家可以输入任何内容并提交。

本关无论输入什么都是错误的。其中有些节点当中存在需要交互的地方，均以粗体标出。

**回合数不可以重置**，每个回合过去了就是过去了，无法重来。（想重来只能新建账号并过至少 5 个题）
